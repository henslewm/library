GLDL Main.vi

This template is for the Queued Message Handler design pattern.

The Event Handling Loop generates messages based on user interface actions.

The Message Handling Loop processes messages generated by the Event Handling Loop, or by other messages. The messages are string values, so new messages can be added easily to the Message Cases case structure in the Message Handling Loop. Each message cluster can also provide an optional value for Message Data, which is a variant that can be converted to whatever message-specific data is required.
